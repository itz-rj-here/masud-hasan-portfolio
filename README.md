# Masud Hasan – Portfolio

Personal portfolio site: Lecturer, Online Educator, Criminologist & Digital Creator.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Run locally

```sh
npm i
npm run dev
```

## Build

```sh
npm run build
npm run preview   # preview production build
```

## Favicon / logo in browser tab

Place your favicon at `public/favicon.ico` (or add a `<link rel="icon" href="/favicon.ico">` in `index.html` pointing to your image). The navbar uses the text “MH” as the site logo; replace or style it in `src/components/Navbar.tsx` if you want an image logo.
